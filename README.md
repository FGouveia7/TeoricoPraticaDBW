# TeoricoPraticaDBW

Este projeto é uma aplicação web desenvolvida para a unidade curricular DBW (Desenvolvimento de Bases Web) da Universidade da Madeira. Demonstra conceitos fundamentais de desenvolvimento web, incluindo HTML, CSS, Bootstrap e JavaScript.

## Funcionalidades

- **Página Inicial**: Apresenta a livraria UMa com uma lista de livros.
- **Página Bootstrap**: Exemplo de utilização do sistema de grelhas do Bootstrap.
- **Página JavaScript**: Exercícios práticos de JavaScript, incluindo templates de string, comparações, manipulação de datas, arrays e objetos.
- **Página de Formulário**: Formulário para interação do usuário.

## Tecnologias Utilizadas

- **Node.js**: Ambiente de execução JavaScript no servidor.
- **Express.js**: Framework web para Node.js.
- **EJS**: Motor de templates para renderização de páginas.
- **Bootstrap**: Framework CSS para design responsivo.
- **JavaScript**: Linguagem de programação para interatividade no lado cliente.

## Instalação

1. Clone o repositório:
   ```
   git clone https://github.com/FGouveia7/TeoricoPraticaDBW.git
   ```

2. Navegue para o diretório do projeto:
   ```
   cd TeoricoPraticaDBW
   ```

3. Instale as dependências:
   ```
   npm install
   ```

## Utilização

1. Inicie o servidor:
   ```
   npm start
   ```

2. Abra o navegador e acesse:
   ```
   http://localhost:3000
   ```

3. Navegue pelas páginas utilizando a navegação no topo.

## Estrutura do Projeto

- `controllers/`: Controladores para gerenciar as rotas.
- `routes/`: Definições das rotas da aplicação.
- `views/`: Templates EJS para as páginas.
- `public/`: Arquivos estáticos (CSS, JS, imagens).
- `Models/`: Modelos de dados (ex: livros).
